
# User Stories
```
As a user
So i can retain information about websites that interest me
I want to see a list of bookmarked links


As a user
So that I can quickly find web sites I recently bookmarked
I would like to see links in descending chronological order


As a user
So i can continue to collect information
I want to add new links

As a user
So i can can organise my bookmarks
I want to add tags to links

As a user
So i can can search for bookmarks by type
I want to filter links by a tag
```
